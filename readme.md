# i2i Academy - Applied Data Science Homework

This repository contains an end-to-end Machine Learning classification pipeline designed to source a dataset, preprocess the features, train multiple classification models, and critically evaluate their performances.

## 📋 Project Pipeline
1. **Data Ingestion:** Sourcing and loading a classification dataset.
2. **Data Preprocessing:** Cleaning missing values, applying Feature Scaling, and executing a stratified Training/Testing split.
3. **Model Selection:** Implementing two distinct classification algorithms from the `scikit-learn` library.
4. **Evaluation:** Extracting comparative Accuracy Scores and Confusion Matrices to analyze predictive patterns.
